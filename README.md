# 🎓 Student Event Registration System

## 📌 Description
A web-based system that allows students to register for events and helps administrators manage event data efficiently.

## ⚙️ Prerequisites
- XAMPP
- PHP
- MySQL

## 🚀 Installation Steps
1. Clone repo
2. Move to htdocs
3. Start Apache & MySQL
4. Import database
5. Run project
